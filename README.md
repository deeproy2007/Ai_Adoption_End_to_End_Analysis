# AI Adoption in Colleges – Data Analytics Project

## Project Summary

This repository contains a **data analytics project** that evaluates the impact of **AI adoption in colleges** on student performance, attendance, and engagement. The detailed analysis, explanations, and interpretations are documented in the **Project Report file** included in this repository.

---

## What This Repository Includes

* **Power BI Dashboard** – Visual insights on GPA, attendance, and AI impact
* **SQL Query File** – Business-driven analytical queries
* **Multiple Datasets** – Student, college, performance, and EdTech usage data
* **Detailed Project Report** – Full documentation of methodology and findings

---

## Key Focus Areas

* GPA comparison before and after AI adoption
* Student engagement with AI-powered EdTech platforms
* Attendance improvement analysis
* Department-wise and semester-wise performance comparison
* Effectiveness of different AI platforms
